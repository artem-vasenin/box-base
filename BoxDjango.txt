1 - Главная
	1.1 - Баннер с кнопкой ведущей на форму обратной связи (page)
		- врхний текст
		- заголовок
		- контент
		- фото
		- ссылка на страницу (контакты например)
	1.2 - О нас с кнопкой ведущей на детальное описание сайта (page)
		- фото
		- краткое описание
		- полный текст
	1.3 - Преимущества (2 столбца текста + фото)
		- Модуль Репитер (заголовок + текст)
		- Фото
	1.4 - Новости (2-3 свежие новости)
		- Модуль новостей из категории с выбором количества
	1.5 - Галерея с просмотром фото
		- Модуль галереи из категории с выбором количества

2 - Новости

		- главная новость (page > accent checkbox || last by date)
		- список новостей (фото + заголовок + просмотры + дата + подробнее)
		- пагинация
	2.3 - Детали новости (page)
		- фото
		- заголовок
		- дата

		- полный текст
		- (галерея фото)
		- (читайте также)

		- название
